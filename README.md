# Prayer Campaigns Mobile App

This app will be a way for users to access prayer campaign content directly on their phone for any prayer campaign.

It will also allow them to set reminders on their phone to remind them to pray for that day's content.

## Apps

### Web

This is to act as a quick prototype so that the concept can be seen and understood by the Prayer Campaigns team.
It could end up being it's own thing, but for now is just a quick prototype.
