# Prayer Campaigns Web Prototype

This is a prototype web application for viewing prayer campaign content. It demonstrates the UI and user flows without API integration.

## Features

- Campaign directory with search and filtering
- Campaign subscription management
- Prayer feed with daily prayer content
- Reminder scheduling UI (no actual notifications)
- Multi-language support (UI only, content is Lorem Ipsum)
- Grayscale theme

## Setup

1. Open `index.html` in a modern web browser
2. No build process or dependencies required

## Usage

- **Campaign Chooser**: Browse and subscribe to campaigns
- **Prayer Feed**: View prayer content for subscribed campaigns
- **Prayer Fuel**: Read daily prayer content
- **Campaigns**: Manage your subscriptions
- **Reminder Config**: Set up reminder schedules (UI only)

## Technical Details

- Pure HTML, CSS, and JavaScript
- No external dependencies
- Uses localStorage for data persistence
- Hash-based routing
- Grayscale theme only
